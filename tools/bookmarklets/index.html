Bookmarklets

What are bookmarklets

Bookmarklets are snippets of JavaScript code that can be used to execute code in the context of the current web page.

How to install

  1. Select the entire content of the bookmarklet code (including the "javascript:" line).
  2. Create a new bookmark in your browser (ideally in the bookmark bar, but it works from anywhere in the browser bookmarks).
  3. Edit the bookmarklet and make sure that it is completely empty.
  4. Paste the bookmarklet code into the bookmark and save it.

How to use

  1. Navigate to the page that you want the code to run on.
  2. Select or click on the bookmarklet.

Table of bookmarklets

Name Where to use What it does
goto-github-page.js

Viewing an HTML file in a GitHub repository.

Redirects user to the same HTML document as it should exist in GitHub Pages. If the page results in an error, GitHub Pages may not be set up or there may be an issue with the deployment on GitHub.com.
goto-github-repo.js

Viewing a web page hosted by GitHub Pages.

Redirects user to the repository that is configured with GitHub Pages.
goto-speed-grader.js

Viewing a "Submission Details" page on Canvas. This is commonly reached by clicking on a link in an email notification from Canvas (that might be something like "Click here to review the submission").

Redirects user to the Speed Grader page for that specific assignment for that student.
grading-assistant.js

Viewing any web page (especially one of the project pages for WDD 130).

Creates a block on information at the top of the page. The information includes colors, fonts, links, etc.

NOTE:
While this bookmarklet tries to ensure that it always displays consistently, some CSS, HTML, JavaScript, etc. that is part of a page can cause it to have some issues. In particular, floating headers can overlap the block. To work around these kinds of issues, set the header (or other piece of HTML) to be hidden ("visibility: none;") using the inpsect tool or do other tweaks as needed.
w3c-css-validator.js

Viewing any web page.

Redirects user to the CSS validator for that page.
w3c-html-validator.js

Viewing any web page.

Redirects user to the HTML validator for that page.
